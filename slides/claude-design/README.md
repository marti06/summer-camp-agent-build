# Claude Design source

The deck was designed in **Claude Design** (claude.ai/design):

- **Project:** "Summer Camp Building Agents"
- **File:** `Build a team agent.dc.html`
- **URL:** https://claude.ai/design/p/93059661-ae52-407a-9db7-6bf915e055f1

`../index.html` is the **implemented** version of that design, the same 22 slides,
extracted into a self-contained deck that runs in any browser with no Claude Design
runtime, no CDN, and no network. That's what you present from.

## Editing the design

- **Quick tweaks** → edit the `<section>`s directly in `../index.html` (each slide
  is plain inline-styled HTML).
- **Visual / drag-and-drop edits** → open the Claude Design project (URL above),
  edit there, then re-run the implement step to refresh `../index.html`.

The Claude Design project is the visual source of record; this repo holds the
runnable implementation.
